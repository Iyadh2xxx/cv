      2. We a group of 2 members together developed a nano processor as a semester end project The pictures of the project are attached herewith

      This was a project where we entirely designed a microprocesor that could read and execute Assembly code to perform simple operations like addition, subtraction, loopinig and negation. Simulataneously displaying the output on a 7 segment display.
      It was coded in VHDL and synthesized using the Vivado platform.
      It is constituted of the following elemental components:

      1. ALU
      2. Register Bank of eight 8 bit registers
      3. 8 bit data bus
      4. Instruction Decoder
      5. Program Counter
      6. 3 bit Instruction bus
